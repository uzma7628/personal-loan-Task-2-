# personal-loan-Task-2-
​"Predicting personal loan acceptance using Machine Learning and Exploratory Data Analysis (EDA)."
​**Data Cleaning**: Sabse pehle null values (missing data) ko check kiya aur unnecessary columns ko drop kiya.
​**Feature Analysis**: Seaborn aur Matplotlib use karke dekha ke Income, Education, aur Family Size ka loan acceptance par kya asar hota hai.
​**Data Preprocessing**: Categorical variables ko numbers mein convert kiya taake Machine Learning model kaam kar sake.
**​Model Building**: Classification model (Logistic Regression) train kiya taake ye predict ho sake ke kaunsa customer loan lega.
**​Evaluation**: Model ki accuracy check ki aur confusion matrix banaya
